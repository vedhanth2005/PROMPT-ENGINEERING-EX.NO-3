# PROMPT ENGINEERING TYPES

---

# 1. Straightforward Prompts

## Definition
Straightforward prompts directly ask the AI to perform a task clearly and concisely without additional context.

## Examples

### Prompt 1
```text
Define photosynthesis in one sentence.
```

### Output
```text
Photosynthesis is the process by which plants use sunlight, water, and carbon dioxide to produce food and oxygen.
```

---

### Prompt 2
```text
List three advantages of electric vehicles.
```

### Output
```text
1. Reduced air pollution
2. Lower fuel costs
3. Environmentally friendly transportation
```

---

## Chatbot Use Case Example

### Prompt
```text
How can I track my product order?
```

### Output
```text
You can track your order using the tracking link sent to your email or by entering your order ID in the "Track Order" section of the website.
```

# 2. Tabular Format Prompting

## Definition
Tabular format prompting requests information in a structured table format for easier understanding and comparison.

## Examples

### Prompt 1
```text
Compare and contrast AC and DC current in a table.
```

### Output

| Feature | AC Current | DC Current |
|---|---|---|
| Full Form | Alternating Current | Direct Current |
| Direction | Changes direction periodically | Flows in one direction |
| Source | Power stations | Batteries |
| Usage | Home appliances | Electronic devices |

---

### Prompt 2
```text
Provide a table listing five programming languages, their paradigms, and one use case each.
```

### Output

| Programming Language | Paradigm | Use Case |
|---|---|---|
| Python | Object-Oriented | Data Science |
| Java | Object-Oriented | Android Apps |
| C | Procedural | System Programming |
| JavaScript | Event-Driven | Web Development |
| R | Statistical | Data Analysis |

---

# 3. Preceding Question Prompting

## Definition
Preceding question prompting asks one question before another to provide context and improve response quality.

## Examples

### Prompt 1
```text
Why is climate change a global concern? Explain how greenhouse gases contribute to global warming.
```

### Output
```text
Climate change is a global concern because it affects weather patterns, ecosystems, and human life. Greenhouse gases trap heat in the atmosphere, causing the Earth’s temperature to rise.
```

---

### Prompt 2
```text
How do vaccines work? Describe the process of immunization in simple terms.
```

### Output
```text
Vaccines train the immune system to recognize and fight harmful viruses or bacteria. Immunization helps the body develop protection against diseases.
```

---

## Chatbot Use Case Example

### Prompt
```text
Why is my internet device not working? Explain the troubleshooting steps.
```

### Output
```text
Your internet device may not work due to loose cables, power issues, or network problems. Restart the router, check cable connections, and verify internet settings.
```

---

# 4. Missing Word Prompting

## Definition
Missing word prompting requires the AI to predict and fill in missing words based on context.

## Examples

### Prompt 1
```text
The capital of France is ____.
```

### Output
```text
Paris
```

---

### Prompt 2
```text
In photosynthesis, plants absorb sunlight to produce ____.
```

### Output
```text
food
```

---

## Chatbot Use Case Example

### Prompt
```text
To reset the router, press the ____ button for 10 seconds.
```

### Output
```text
reset
```
